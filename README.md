# Object-Oriented Programming (OOP)

This repository is created to build a strong and clear understanding of **Object-Oriented Programming (OOP)** concepts through simple explanations and practical code examples. It is suitable for beginners, students, and anyone revising OOP fundamentals.

---

##  What is Object-Oriented Programming?

Object-Oriented Programming is a programming paradigm that organizes code into **objects**, which are instances of **classes**. Each object contains data (attributes) and behavior (methods). OOP helps in writing clean, modular, reusable, and maintainable code.
